# JOSE & Related Standards

- **JWS** (JSON Web Signature) – signed token.
- **JWE** (JSON Web Encryption) – encrypted token.
- **JWK** (JSON Web Key) – key representation.
- **JWA** (JSON Web Algorithms) – algorithm definitions.
- **JWT** – the token format.
